# Meridian-Schedule

Meridian-Schedule is a structured scheduling system within Meridian for organizing work, research, and everyday life.

## Overview

This project is designed as a personal planning system that connects daily schedules, tasks, priorities, and weekly organization.

## Core Areas

- daily scheduling
- task management
- weekly planning
- domain-based organization
- long-term personal workflow

## Planned Modules

- Dashboard
- Calendar
- Tasks
- Weekly Planning
- Notes

## Project Structure

- `docs/` – project notes and planning documents
- `src/` – source code
- `components/` – reusable UI components
- `pages/` – page-level structure
- `data/` – mock data and future models
- `types/` – shared type definitions
