# Features

## Planned Features
- Dashboard
- Calendar view
- Task list
- Weekly planning
- Notes
- Domain filters

## Core Modules

### Dashboard
A central overview of tasks, priorities, and upcoming items.

### Calendar
Daily and weekly schedule views for time-based planning.

### Tasks
Task management with status, deadline, priority, and category.

### Weekly Planning
A space for weekly priorities, focus, and review.

### Notes
A lightweight area for reflections and temporary planning context.
