# MVP

## First Version Goals

The first version of Meridian-Schedule will focus on a lightweight personal scheduling workflow.

### Core features
- add tasks
- assign date and time
- view today's schedule
- view weekly plans
- mark tasks as complete
- group tasks by domain

## Task Fields
- title
- description
- date
- start_time
- end_time
- priority
- status
- domain
- notes
